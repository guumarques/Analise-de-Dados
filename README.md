# Análise de Dados com Python

## Sobre o Repositório

Este repositório reúne exercícios e atividades desenvolvidas durante a disciplina de Análise de Dados da faculdade.

Além das listas de exercícios práticas, o repositório também contém um seminário envolvendo análise de dados da empresa **Tesla**, com foco em informações relacionadas às entregas de veículos e interpretação de dados.

---

## Tecnologias Utilizadas

- Python
- Matplot
- Seaborn
- Pandas
- NumPy

---

## Conteúdo do Repositório

### Exercícios
Exercícios voltados para prática de:
- Manipulação de dados
- Limpeza e tratamento de datasets
- Estatística básica
- Análise exploratória de dados (EDA)
- Criação de gráficos e visualizações

### 📊 Seminário
Análise de dados da Empresa Tesla envolvendo:
- Quantidade de entregas de veículos
- Tendências e crescimento ao longo do tempo
- Interpretação de gráficos
- Visualização de dados

---

## Como Executar

Clone o repositório:

```bash
git clone [<url-do-repositorio>](https://github.com/guumarques/Analise-de-Dados.git)
```

Instale as dependências:
 - Todas as dependências estarão nos arquivos jupyter.


Execute o Jupyter Notebook:

```bash
jupyter notebook
```
